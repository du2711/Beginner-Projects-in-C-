This repository documents my journey of learning **C programming** from the fundamentals to intermediate concepts. It contains topic-wise programs, practice questions, and implementations of core programming concepts that helped me build a strong programming foundation.

After completing topics up to **Sorting Algorithms**, I transitioned to **C++** to continue my journey in **Data Structures & Algorithms (DSA)** and competitive programming.

Although this repository is no longer actively updated, it serves as a reference for beginners learning C.

## Running the Programs

Compile:

```bash
gcc filename.c -o filename
```

Run (Linux/macOS):

```bash
./filename
```

Run (Windows):

```bash
filename.exe
```
